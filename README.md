# Mixed Reality專案
利用基於混和現實（Mixed Reality, MR）的紅綠燈交通系統,透過Unity結合Meta Quest 3裝置，模擬真實交通場景

5. [使用流程文件](#使用流程文件)
## 前置需求
* Meta Quest裝置
* Unity (2022 以上版本)
* Visual Studio
## 檔案
下載整個檔案後將APK檔案中(passthrough mode texting.exe)載入Meta Quest裝置中即可運行
